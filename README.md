# go-hn

> Simple command-line util to fetch stories from Hacker News

## Usage

```
Usage:

	$ hn [flags] <command>

Commands:

	top		Display top posts
	new		Display new posts
	best		Display best posts

Flags:

	--limit		Cap max results

Examples:

	$ hn --limit 3 top
	$ hn best
```
